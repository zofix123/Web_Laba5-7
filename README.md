### **Сайт знакомств (учебный проект)**
В ходе разработки будет использоваться язык Java для бэкенда, база данных PostgeSQL для хранения информации о пользователях, а также технологии Spring Boot, Maven, Thymeleaf.

	- [x] реализовать кабинет пользователя с возможностью загрузки изображения на портал;
	- [x] реализовать счетчик посещений и отображение текущей даты и времени, запрашиваемых с сервера. Счетчик должен сохранять свое состояние после перезагрузки сервера;
	- [x] разработать набор классов и страниц для аутентификации пользователя и хранении информации в сессии;
2. **Реализация функционала кабинета администратора (6 лабораторная)****
	- [x] разработать набор классов и страниц для реализации базового функционала пользователя, оператора и администратора в системе согласно предметной области;
	- [x] реализовать кабинет администратора. Реализовать возможность просмотра и редактирования данных пользователей, назначение ролей;
3. **Реализация новостной ленты портала (7 лабораторная)****
	- [ ] разработать набор классов и страниц для реализации новостной ленты согласно предметной области.
	- [ ] реализовать кабинет модератора. Реализовать возможность добавления и редактирования новостей
